Introduction

Semantics in HTML refers to the meaning of the elements and how they convey information about the content.

Importance of Semantics

Semantic HTML improves accessibility, SEO, and maintainability of web pages.

Examples of Semantic Elements

Some common semantic elements include <header>, <footer>, <article>, and <section>.
